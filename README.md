# Js

> 해당 레파지토리는 js의 객체의 개념을 공부하기 위함이다. 여기서, 우리는 함수형이란 무엇인지 고민을 해보면서 코드를 작성할 것이다.

## 내가 생각하는 형(style,syntax,approach)이란 형태

클래스형, 함수형은 단순 하나의 수단일 뿐이다. 우리는 객체를 바라봐야하고 새로운 수단을 성립하거나 적용할 수 있는 객체지향적 사고를 가져야한다. 이곳이 바로 그런 사고의 훈련을 하기 위해 존재한다.

## infra

AWS의 S3로 HTML을 배포한다. 정적인 HTML이기 때문에 많은 준비를 하지 않아도 된다. 따라서, build 없이, webpack이나 bable없이 배포를 해도 괜찮다. 이와 같이 구성한 이유는 정적인 도메인이 무엇인지 알아가며, webpack과 bable이 필요한 환경을 직접 느껴보며, 이를 점차 고도화하는 작업을 해보려고 한다.

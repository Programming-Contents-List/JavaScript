# Section1

## 목표

객체를 생성하고 key값을 관리할 수 있는 로직을 구현한다.

## 구현

- [리터럴 객체](./object/literal/generateElement.js)

## 문제점

현재 코드는 은닉성이 보장되지 않는다. 리터럴로 구현된 로직은 set이나 객체의 프로퍼티로 접근해서 외부에서 수정을 할 수 있게 된다. 이를 해결하기 위해서 함수형으로 변경을 하고 관리할 수 있게 코드를 구현해야한다.

## 의문과 숙제

왜 변수는 접근을 할 수가 없는지, 캡슐화가 무엇인지, 프로퍼티를 유지하되 은닉성을 보장 할 수 있게 구현할 수 없는지.
